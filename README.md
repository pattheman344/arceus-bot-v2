# Arceus Bot V2

# How to install

## Make sure you have the following python modules installed on your system

### discord.py

### random

### praw

### json

### asyncio

### os

# Setting up the meme command

## In order to get the meme command working, you have to enter your reddit credentials, to get your client id, client secret and user agent, head over to https://www.reddit.com/prefs/apps and create an app.



### Pull requests are welcome. make sure to fill out the config.json in order to get the bot working and to install all the modules above.
